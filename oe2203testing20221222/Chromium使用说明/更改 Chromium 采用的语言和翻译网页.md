# 更改 Chromium 采用的语言和翻译网页  
在 Chrome 中翻译网页

当您浏览的网页使用的是您看不懂的语言时，您可以使用 Chrome 翻译该网页。

    在计算机上打开 Chrome。
    转到一个不是以您要翻译成的目标语言显示的网页。
    在地址栏右侧点击“翻译”图标 。
    Chrome 即会翻译当前网页。

无法正常翻译？请尝试刷新该网页。如果仍然无法正常翻译，请右键点击该页面中的任意位置，然后点击翻成[相应语言]。
更改默认的网页翻译设置

默认情况下，Chrome 会询问是否翻译非您所用语言的网页。

更改 Chrome 浏览器采用的语言

您可对 Chrome 进行设置，让其以您想使用的语言显示所有的设置和菜单。此选项仅适用于 Windows 计算机。

重要提示：若要在 Chromebook 上添加或移除网络内容语言，请了解如何管理语言。

您使用的是 Mac 或 Linux 系统？ Chrome 会自动以您计算机的默认系统语言显示相应内容。

    在计算机上打开 Chrome。
    在右上角，依次点击“更多”图标 展开 然后 设置。
    点击底部的高级。
    依次点击语言 然后 语言。
    在您要使用的语言旁边，点击“更多”图标 展开。
        如果没有列出要使用的语言，请点击添加语言来添加该语言。
    点击以这种语言显示 Google Chrome。 
        此选项仅适用于 Windows 计算机。
    重启 Chrome 以应用所做的更改。

输入不受支持的语言

如果您需要输入您的计算机不支持的语言，可以使用 Google 输入工具这一 Chrome 扩展程序。

Windows 用户可能需要配置自己的 Windows 语言设置，才能添加东亚语言或其他使用复杂文字的语言。
